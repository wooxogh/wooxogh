## 🧠 About Me

백엔드 구조 설계를 통해 **AI 서비스의 병목을 제거하는 개발자**입니다.  

- 비동기 처리 / 캐싱 / 서비스 분리를 통해 **응답 속도 개선**
- “기능 구현”보다 **병목을 구조적으로 해결하는 것**에 집중
- 실험이 아닌 **실제 동작하는 시스템**을 만드는 것을 지향합니다
